# ФАЙЛИ В СТИЛІ С

[![stdio.h](https://img.shields.io/badge/C-stdio.h-blue.png?style=flat-square&logoColor=white)](https://en.cppreference.com/c/header/stdio)

## Мета

Оволодіння практичними навичками роботи з файлами в мові С.

## Функції

Головна стандартна бібліотека для файлів у стилі C це `<stdio.h>`. Вона тримає тип `FILE` і основні функції
файлового введення та виведення. Найбільш поширені у використанні:

- відкриття і закриття 
- форматоване читання і запис
- символьне читання і запис
- рядкове читання і запис
- бінарне читання і запис
- позиціонування, перевірка стану потоку

### Порівняння C vs C++

| C стиль     | C++ стиль                                        |
|-------------|--------------------------------------------------|
| `<stdio.h>` | `<fstream>`                                      |
| `FILE*`     | `std::ifstream`, `std::ofstream`, `std::fstream` |
| `fopen()`   | конструктор потоку або `.open()`                 |
| `fprintf()` | `<<`                                             |
| `fscanf()`  | `>>`                                             |
| `fclose()`  | `.close()`                                       |

### Приклад C стилю:

```c++
#include <stdio.h>

FILE* f = fopen("test.txt", "w");
fprintf(f, "Hello");
fclose(f);
```

### Приклад C++ стилю:

```c++
#include <fstream>

std::ofstream file("test.txt");
file << "Hello";
file.close();
```

## Відкриття і закриття файлу

Будь яка робота з файлом у `C` починається з відкриття через `fopen()` 
і завершується закриттям через `fclose()`. 
Файл представляється покажчиком типу `FILE*`.

| Функція  | Сигнатура                                              | Параметри                                   | Призначення                               |
|----------|--------------------------|--------------------------|--------------------------|
| `fopen`  | `FILE* fopen(const char* filename, const char* mode);` | `filename` ім’я файлу, `mode` режим доступу | Відкриває файл і повертає файловий потік. |
| `fclose` | `int fclose(FILE* stream);`                            | `stream` файловий потік                     | Закриває файл і звільняє ресурси.         |

Приклад:

```c++
FILE* f = fopen("test.txt", "w");
if (f == NULL) return 1;
fclose(f);
```

| Режим | Значення                    | Для чого використовується                          |
|-------|-----------------------------|----------------------------------------------------|
| `r`   | тільки читання              | Читання існуючого текстового файлу.                |
| `w`   | тільки запис                | Створення нового файлу або повне очищення старого. |
| `a`   | дозапис у кінець            | Додавання нових даних без стирання старих.         |
| `rb`  | binary read                 | Читання бінарних даних без перетворення байтів.    |
| `wb`  | binary write                | Запис сирих байтів у двійковому форматі.           |
| `ab`  | binary append               | Дозапис двійкових даних у кінець файлу.            |
| `r+`  | читання і запис             | Редагування існуючого файлу.                       |
| `w+`  | читання і запис з очищенням | Повне перезаписування файлу.                       |

- Особливості:
  - `FILE*` це покажчик на файловий потік
  - `NULL` означає помилку відкриття
  - `rb/wb` важливі для бінарних файлів
  - у Windows текстовий режим може змінювати `\n`
  - файл треба закривати через `fclose()`

## Форматований запис і читання

Функції `fprintf()` і `fscanf()` працюють аналогічно `printf()` і `scanf()`,
але з файлом замість консолі. Вони використовуються для текстових даних
і форматованого вводу виводу.

| Функція   | Сигнатура                                             | Параметри                      | Призначення                        |
|----------|--------------------------|--------------------------|--------------------------|
| `fprintf` | `int fprintf(FILE* stream, const char* format, ...);` | `stream` файл, `format` шаблон | Записує форматований текст у файл. |
| `fscanf`  | `int fscanf(FILE* stream, const char* format, ...);`  | `stream` файл, `format` шаблон | Читає форматовані дані з файлу.    |

Приклад:

```c++
FILE* f = fopen("data.txt", "w");
fprintf(f, "%s %d", "Age", 20);
fclose(f);

f = fopen("data.txt", "r");
char word[20];
int x;
fscanf(f, "%s %d", word, &x);
printf("%s %d\n", word, x);
fclose(f);
```

| Специфікатор | Тип даних | Приклад   |
|--------------|-----------|-----------|
| `%d`         | `int`     | `25`      |
| `%f`         | `float`   | `3.14`    |
| `%lf`        | `double`  | `2.718`   |
| `%c`         | `char`    | `A`       |
| `%s`         | рядок     | `"Hello"` |

- Особливості:
  - працює тільки з текстовими даними
  - `%s` читає до пробілу
  - для читання потрібні адреси змінних
  - `fprintf()` повертає кількість записаних символів
  - `fscanf()` повертає кількість успішно прочитаних значень

## Посимвольна робота з файлом

Функції `fgetc()` і `fputc()` читають або записують один символ за раз.
Використовуються для парсерів, текстових аналізаторів і низькорівневої обробки тексту.

| Функція | Сигнатура                          | Параметри                  | Призначення                 |
|----------|--------------------------|--------------------------|--------------------------|
| `fgetc` | `int fgetc(FILE* stream);`         | `stream` файл              | Читає один символ з файлу.  |
| `fputc` | `int fputc(int ch, FILE* stream);` | `ch` символ, `stream` файл | Записує один символ у файл. |

Приклад:

```c++
FILE* f = fopen("chars.txt", "w");
fputc('A', f);
fputc('B', f);
fclose(f);

f = fopen("chars.txt", "r");
char ch;
ch = fgetc(f);
printf("%c\n", ch);
ch = fgetc(f);
printf("%c\n", ch);
fclose(f);
```

| Значення | Опис                              |
|----------|-----------------------------------|
| `EOF`    | Кінець файлу або помилка читання. |
| `ASCII`  | Код символу у таблиці ASCII.      |


- Особливості:
  - `fgetc()` повертає `int`, а не `char`
  - `EOF` зазвичай дорівнює `-1`
  - підходить для парсерів і лексерів
  - повільніше за блокове читання
  - дозволяє обробляти файл посимвольно

## Робота з рядками

Функції `fgets()` і `fputs()` працюють з цілими рядками.
Стандартний спосіб безпечної роботи з текстовими файлами у `C`.

| Функція | Сигнатура                                         | Параметри                                 | Призначення           |
|----------|--------------------------|--------------------------|--------------------------|
| `fgets` | `char* fgets(char* str, int size, FILE* stream);` | `str` буфер, `size` розмір, `stream` файл | Читає рядок з файлу.  |
| `fputs` | `int fputs(const char* str, FILE* stream);`       | `str` рядок, `stream` файл                | Записує рядок у файл. |

Приклад:

```c++
fputs("Hello\n", f);
fputs("World\n", f);
fclose(f);

f = fopen("text.txt", "r");
char str[100];
fgets(str, 100, f);
printf("%s", str);
fclose(f);
```

| Параметр | Значення                                    |
|----------|---------------------------------------------|
| `str`    | Масив символів для рядка.                   |
| `size`   | Максимальна кількість символів для читання. |
| `stream` | Файловий потік.                             |

- Особливості:
  - `fgets()` 
    - читає максимум `size - 1` символів
    - додає `\0` у кінець рядка
    - може зчитати `\n`
    - зупиняється на `\n`, `EOF` або ліміті `size - 1`
  - `fputs()` 
    - записує рядок як є
    - не додає `\n` автоматично
    - повертає не від’ємне число при успіху
    - безпечніше за `gets()`

## Бінарна робота з файлами

Функції `fread()` і `fwrite()` працюють з сирими байтами пам’яті.
Вони використовуються для масивів, структур, зображень, баз даних
і будь-яких двійкових форматів.

| Функція  | Сигнатура                                                                  | Параметри                                                             | Призначення                 |
|----------|--------------------------|--------------------------|--------------------------|
| `fread`  | `size_t fread(void* ptr, size_t size, size_t count, FILE* stream);`        | `ptr` буфер, `size` розмір елемента, `count` кількість, `stream` файл | Читає блок байтів з файлу.  |
| `fwrite` | `size_t fwrite(const void* ptr, size_t size, size_t count, FILE* stream);` | `ptr` дані, `size` розмір елемента, `count` кількість, `stream` файл  | Записує блок байтів у файл. |

Приклад:

```c++
FILE* f = fopen("nums.dat", "wb");
int arr[3] = {10, 20, 30};
fwrite(arr, sizeof(int), 3, f);
fclose(f);

f = fopen("nums.dat", "rb");
int arr2[3];
fread(arr2, sizeof(int), 3, f);

for (int i = 0; i < 3; i++)
    printf("%d\n", arr2[i]);

fclose(f);
```

| Параметр | Значення                         |
|----------|----------------------------------|
| `ptr`    | Адреса даних або буфера.         |
| `size`   | Розмір одного елемента у байтах. |
| `count`  | Кількість елементів.             |
| `stream` | Файловий потік.                  |

- Особливості:
  - `rb/wb` потрібні для двійкових даних
  -  у текстовому режимі байти можуть змінюватися
  - `sizeof()` дозволяє правильно рахувати байти
  - підходить для структур і масивів
  - `fread()` і fwrite()` працюють напряму з пам’яттю

## Позиціонування у файлі

Функції позиціонування дозволяють переміщувати поточну позицію читання або запису у файлі.
Це потрібно для довільного доступу до даних без читання файлу повністю.

| Функція  | Сигнатура                                           | Параметри                                                | Призначення                        |
|----------|--------------------------|--------------------------|--------------------------|
| `fseek`  | `int fseek(FILE* stream, long offset, int origin);` | `stream` файл, `offset` зміщення, `origin` точка відліку | Переміщує позицію у файлі.         |
| `ftell`  | `long ftell(FILE* stream);`                         | `stream` файл                                            | Повертає поточну позицію у байтах. |
| `rewind` | `void rewind(FILE* stream);`                        | `stream` файл                                            | Повертає позицію на початок файлу. |

Приклад:

```c++
FILE* f = fopen("nums.dat", "rb");
fseek(f, 5 * sizeof(int), SEEK_SET);
int x;
fread(&x, sizeof(int), 1, f);
printf("%d\n", x);
fclose(f);
```

| Константа  | Значення        | Для чого використовується      |
|------------|-----------------|--------------------------------|
| `SEEK_SET` | початок файлу   | Перехід від початку файлу.     |
| `SEEK_CUR` | поточна позиція | Зміщення від поточної позиції. |
| `SEEK_END` | кінець файлу    | Перехід від кінця файлу.       |

- Особливості:
  - `fseek()` працює у байтах
  - `offset` може бути додатним або від’ємним
  - `ftell()` часто використовується для визначення розміру файлу
  - `rewind()` еквівалентний `fseek(stream, 0, SEEK_SET)`
  - позиціонування особливо важливе для бінарних файлів

## Перевірка стану потоку

Функції перевірки стану дозволяють визначати помилки читання, запису
або досягнення кінця файлу.
Вони використовуються для контролю коректності файлових операцій.

| Функція    | Сигнатура                      | Параметри     | Призначення                         |
|------------|--------------------------------|---------------|-------------------------------------|
| `feof`     | `int feof(FILE* stream);`      | `stream` файл | Перевіряє досягнення кінця файлу.   |
| `ferror`   | `int ferror(FILE* stream);`    | `stream` файл | Перевіряє наявність помилки потоку. |
| `clearerr` | `void clearerr(FILE* stream);` | `stream` файл | Скидає прапори помилки і EOF.       |

```c++
FILE* f = fopen("text.txt", "r");

char ch;

while ((ch = fgetc(f)) != EOF)
  putchar(ch);

if (feof(f))
  printf("Кінець файлу\n");

if (ferror(f))
  printf("Помилка читання\n");

fclose(f);
```

| Стан         | Значення                  |
|--------------|---------------------------|
| `EOF`        | Досягнуто кінець файлу.   |
| `error flag` | У потоці виникла помилка. |

- Особливості:
  - `feof()` спрацьовує тільки після невдалого читання
  - `EOF` не завжди означає помилку
  - `ferror()` використовується для перевірки помилок диска або потоку
  - `clearerr()` дозволяє продовжити роботу з потоком
  - перевірка стану особливо важлива при циклічному читанні файлів
