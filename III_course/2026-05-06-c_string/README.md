## Що таке рядки в стилі С?

Рядок у **стилі C** це масив символів типу char, який завершується нульовим символом `'\0'`.
Цей символ повідомляє функціям стандартної бібліотеки, де закінчується рядок.
Якщо у масиві немає `'\0'`, то функції на кшталт `strlen`, `strcpy`, `cout << char*` або `puts`
будуть читати пам’ять далі, поки випадково не зустрінуть нульовий байт. 
Через це рядок на `N` видимих символів потребує масив розміром `N + 1`.

| Конструкція     | Значення                                             |
|-----------------|------------------------------------------------------|
| `char str[11];` | Масив для рядка максимум із 10 символів і `'\0'`     |
| `'\0'`          | Нульовий символ, маркер кінця рядка                  |
| `char* p`       | Покажчик на перший символ рядка                      |
| `strlen(str)`   | Довжина без урахування `'\0'`                        |
| `sizeof(str)`   | Розмір усього масиву в байтах, якщо `str` саме масив |

Приклад:

```cpp
char str[] = "Hello";
cout << str << endl;
cout << strlen(str) << endl;
cout << sizeof(str) << endl;
```

У цьому прикладі `str` містить не п’ять, а шість елементів: `H, e, l, l, o, '\0'`. Функція `strlen(str)` повертає `5`, 
бо рахує тільки видимі символи до нульового байта. Оператор `sizeof(str)` повертає `6`, 
бо враховує весь масив разом із завершальним символом.

## Як відбувається ініціалізація рядка?

Рядок у **стилі C** зберігається як послідовність байтів. Кожен символ займає одну комірку типу `char`, 
а останній елемент повинен містити `'\0'`. Тому в `C style strings` немає окремого поля довжини. 
Довжина визначається тільки проходом по пам’яті до першого нульового символу.

| Представлення                      | Пам’ять                            |
|------------------------------------|------------------------------------|
| `"Hello"`                          | `H e l l o \0`                     |
| `"ABC"`                            | `A B C \0`                         |
| `char str[8] = "Hi";`              | `H i \0 ? ? ? ? ?`                 |
| `char str[] = {'A','B','C'};`      | Не є коректним C рядком без `'\0'` |
| `char str[] = {'A','B','C','\0'};` | Коректний C рядок                  |

Приклад:

```cpp
char correct[] = {'A', 'B', 'C', '\0'};
char incorrect[] = {'A', 'B', 'C'};

cout << correct << endl;
cout << incorrect << endl;
```

Перший масив є коректним рядком у **стилі C**, бо він явно завершується `'\0'`. 
Другий масив є просто масивом символів, але не надійним рядком. 
Його друк через `cout` може вивести зайві символи, бо потік буде читати пам’ять далі, поки не знайде нульовий байт.

## Як визначити довжину рядка?
   
Довжину рядка **стилі C** визначають за допомогою функції `strlen()`. Вона рахує кількість символів від початку 
рядка до першого нульового символу `'\0'`. Сам нульовий символ у довжину не входить. 
Важливо розрізняти `strlen()` і `sizeof()`: `strlen()` показує фактичну довжину тексту, 
а `sizeof()` показує розмір масиву в байтах, якщо змінна є саме масивом, а не покажчиком.

| Конструкція       | Значення                                                        |
|-------------------|-----------------------------------------------------------------|
| `strlen(str)`     | Кількість символів до `'\0'`                                    |
| `sizeof(str)`     | Повний розмір масиву в байтах                                   |
| `strlen(str) + 1` | Кількість байтів, потрібна для копіювання рядка разом із `'\0'` |
| `sizeof(char*)`   | Розмір покажчика, а не рядка                                    |
| `str[i] != '\0'`  | Умова ручного проходу по рядку                                  |

   
Приклад:
 
```cpp
char str[20] = "qwerty";
char* p = str;

cout << strlen(str) << endl;
cout << sizeof(str) << endl;
cout << sizeof(p) << endl;
```  

У цьому прикладі `strlen(str)` поверне `6`, бо рядок містить шість видимих символів. 
`sizeof(str)` поверне `20`, бо `str` є масивом на `20` елементів типу `char`. А `sizeof(p)`
поверне розмір покажчика, наприклад `8` байтів у `64` бітній системі, 
тому що `p` зберігає лише адресу першого символу, а не весь рядок.

## Навести приклади звернення до елементів рядка.

Оскільки рядок у **стилі C** є масивом символів, до його елементів можна звертатися через індекси
або через покажчики. Запис `str[i]` означає доступ до символу з індексом `i`, а запис `*(str + i)` означає те саме, 
але через адресну арифметику. Покажчик на `char` можна зміщувати по рядку, оскільки кожен символ займає один байт.

| Конструкція        | Значення                                |
|--------------------|-----------------------------------------|
| `str[0]`           | Перший символ рядка                     |
| `*str`             | Перший символ рядка через покажчик      |
| `str[1]`           | Другий символ рядка                     |
| `*(str + 1)`       | Другий символ рядка через покажчик      |
| `str + 1`          | Адреса другого символу                  |
| `cout << str`      | Друк усього рядка до `'\0'`             |
| `cout << str + 1`  | Друк рядка, починаючи з другого символу |
| `str[i] = 'A'`     | Зміна символу за індексом               |
| `*(str + i) = 'A'` | Зміна символу через покажчик            |

Приклад:

```cpp
char str[] = "PRIVET";

cout << str << endl;
cout << *str << endl;
cout << *(str + 1) << endl;
cout << str + 1 << endl;

*(str + 2) = 'A';

cout << str << endl;

for (int i = 0; i < strlen(str); i++)
cout << str[i] << ' ';
```

У цьому прикладі `cout << str` виводить увесь рядок `PRIVET`. Вираз `*str` виводить тільки перший символ `P`, 
а `*(str + 1)` виводить другий символ `R`. Вираз `cout << str + 1` виводить не адресу, 
а частину рядка, починаючи з другого символу, тобто `RIVET`. Після виконання `*(str + 2) = 'A'`
третій символ змінюється, і рядок стає `PRAVET`.

## Якою функцією можна ввести рядок з пропусками з екрану? Навести приклад.

У чистому **C стилі** рядок з пропусками можна ввести за допомогою функції `fgets()`.
На відміну від `scanf("%s", str)`, яка читає тільки до першого пропуску, функція `fgets()`
читає весь рядок разом із пропусками до символу нового рядка або до заповнення буфера.

| Спосіб введення                 | Приклад                           | Особливість                             |
|---------------------------------|-----------------------------------|-----------------------------------------|
| `cin >> str`                    | `cin >> str;`                     | Читає тільки до першого пропуску        |
| `cin.getline(str, size)`        | `cin.getline(str, 80);`           | Читає рядок із пропусками               |
| `cin.getline(str, sizeof(str))` | `cin.getline(str, sizeof(str));`  | Розмір береться з масиву                |
| `fgets(str, size, stdin)`       | `fgets(str, sizeof(str), stdin);` | C style варіант із контролем розміру    |
| `gets(str)`                     | `gets(str);`                      | Небезпечна функція, не використовується |


Приклад:

```cpp
char str[80];

printf("Введіть речення: ");
fgets(str, sizeof(str), stdin);

str[strcspn(str, "\n")] = '\0';

printf("Введений рядок: %s\n", str);
printf("Довжина рядка: %zu\n", strlen(str));
```

У цьому прикладі `fgets()` читає речення з клавіатури разом із пропусками. 
Другий аргумент `sizeof(str)` обмежує кількість символів, які можуть бути записані в масив. 
Це потрібно, тому що рядки в **стилі C** не мають автоматичного контролю розміру буфера. 
Після введення функція `fgets()` може залишити в кінці рядка символ нового рядка `\n`, 
тому рядок `str[strcspn(str, "\n")] = '\0';` замінює його на нульовий символ завершення рядка.

## Як ввести та вивести один символ?

У чистому **C стилі** один символ можна ввести за допомогою функції `getchar()`, а вивести за допомогою 
функції `putchar()`. Символ має тип char і є окремим елементом символьного масиву. 
Якщо рядок у **стилі C** є масивом символів, то один символ є значенням однієї комірки цього масиву.

| Функція       | Призначення                                             |
|---------------|---------------------------------------------------------|
| `getchar()`   | Введення одного символу зі стандартного потоку введення |
| `putchar(ch)` | Виведення одного символу у стандартний потік виведення  |
| `toupper(ch)` | Перетворення символу у верхній регістр                  |
| `tolower(ch)` | Перетворення символу у нижній регістр                   |

Приклад:

```cpp
char ch;

printf("Введіть символ: ");
ch = getchar();

printf("Введений символ: ");
putchar(ch);
putchar('\n');

char str[] = "c language";

for (int i = 0; str[i] != '\0'; i++)
putchar(toupper(str[i]));

putchar('\n');
```

У цьому прикладі `getchar()` читає один символ із клавіатури, а `putchar(ch)` виводить його на екран.
Далі програма проходить по рядку `str` до нульового символу `'\0'` і виводить кожен символ через `putchar()`,
попередньо перетворюючи його у верхній регістр за допомогою `toupper()`.

## Якою функцією можливо скопіювати один рядок в другий?

У чистому **C стилі** для копіювання одного рядка в інший використовують функцію `strcpy()`. 
Вона копіює символи з рядка джерела в цільовий масив разом із завершальним нульовим символом `'\0'`. 
Цільовий масив повинен мати достатній розмір, інакше може статися вихід за межі буфера. 
Для обмеженого копіювання використовують `strncpy()`, але після нього потрібно контролювати 
наявність завершального `'\0'`.

| Функція                | Призначення                                         |
|------------------------|-----------------------------------------------------|
| `strcpy(dst, src)`     | Копіює весь рядок `src` у `dst`                     |
| `strncpy(dst, src, n)` | Копіює не більше `n` символів                       |
| `strlen(src) + 1`      | Мінімальний розмір буфера для копії разом із `'\0'` |
| `dst[n] = '\0'`        | Ручне завершення рядка після часткового копіювання  |

Приклад:

```cpp
char source[] = "QWERTY";
char destination[20];

strcpy(destination, source);

printf("%s\n", destination);
```

У цьому прикладі рядок source копіюється в масив `destination`. Оскільки `destination` має розмір `20` символів, 
у ньому достатньо місця для шести символів `QWERTY` і завершального нульового символу `'\0'`.

## Як порівняти два рядки до певної кількості символів?

У чистому **C стилі** для порівняння двох рядків до певної кількості символів використовують функцію `strncmp()`. 
Вона порівнює не весь рядок, а тільки перші `n` символів. Порівняння виконується посимвольно за кодами символів.

| Функція              | Призначення                    |
|----------------------|--------------------------------|
| `strcmp(s1, s2)`     | Порівнює два рядки повністю    |
| `strncmp(s1, s2, n)` | Порівнює перші `n` символів    |
| Результат `0`        | Рядки або частини рядків рівні |
| Результат `< 0`      | Перший рядок менший            |
| Результат `> 0`      | Перший рядок більший           |


Приклад:

```cpp
printf("%d\n", strcmp("abc", "abc"));
printf("%d\n", strcmp("abc", "abd"));

printf("%d\n", strncmp("aa5566", "aa7", 2));
printf("%d\n", strncmp("abcdef", "abcxyz", 3));
```

У цьому прикладі `strcmp("abc", "abc")` поверне `0`, бо рядки однакові. 
`strcmp("abc", "abd")` поверне від’ємне значення, бо символ `c` менший за `d`. Виклик `strncmp("aa5566", "aa7", 2)`
поверне `0`, бо перші два символи в обох рядках однакові. Так само `strncmp("abcdef", "abcxyz", 3)` поверне `0`, 
бо порівнюються тільки перші три символи `abc`.
