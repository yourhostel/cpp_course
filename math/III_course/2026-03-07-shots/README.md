# Практичне заняття No2

## Визначення ймовірності подій

### Мета:

- Навчитися розв'язувати задачі з теорії ймовірностей у програмному коді `C++`.
  Ознайомитися із застосуванням шаблонних функцій та механізму обмеження типів `concepts` на прикладі обчислення ймовірнісних характеристик.

### Завдання:
- При стрільбі із гвинтівки відносна частота влучення в мішень дорівнює 0,85. Скільки буде влучень, якщо проведено 120 пострілів? [17]

### Розв'язання

- Функція обчислення повинна реалізувати математичне сподівання кількості влучень у схемі Бернуллі. Якщо ймовірність влучення при одному пострілі дорівнює $p$, а кількість пострілів $n$, тоді очікувана кількість влучень:

$$
E(X) = n \cdot p
$$

- Для перевірки коректності введених даних використаємо шаблонну функцію з обмеженнями типів `concepts`.

```cpp
template<std::integral ShotType, std::floating_point ProbType>
bool validate_input(ShotType shots, ProbType probability)
```

### Висновок: 

Опрацьовано використання шаблонної функції як механізму перевірки коректності введених даних перед виконанням основного алгоритму обчислення. Для цього застосовано параметри типів шаблону ShotType та ProbType, семантичні назви яких відображають їх призначення у програмі. Параметр ShotType використовується для представлення типу кількості пострілів (shot означає постріл), а ProbType для представлення типу ймовірності (prob є скороченням від probability). Обмеження типів реалізовано за допомогою механізму concepts, що дозволяє гарантувати використання цілого типу для кількості пострілів та типу з плаваючою комою для значення ймовірності. Додатково використано засіб стандартної бібліотеки std::numeric_limits для контролю допустимого діапазону значень цілого типу та запобігання переповненню.
